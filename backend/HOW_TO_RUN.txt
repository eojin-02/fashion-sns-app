    # 방법 B: 데이터 유지하고 마이그레이션만
    docker exec -it fsns-db psql -U fsns -d fsns -c "ALTER TABLE users ADD COLUMN avatar_bundle_key VARCHAR(255); ALTER TABLE users ADD COLUMN avatar_config JSONB;"


2.백엔드 스프링
    .\gradlew.bat bootRun

    application.yml 기본값이 전부 localhost를 가리켜서 환경변수 없이 바로 뜹니다 (포트 8080).
    구글 로그인을 실제로 쓰려면 먼저: $env:APP_GOOGLE_CLIENT_ID = "<웹 클라이언트 ID>" 설정 후 bootRun. 없어도 dev-login으로 개발 가능합니다.
    뜨면 http://localhost:8080/swagger-ui.html 에서 API 확인 가능.
